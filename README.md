# Al Aman Gym Website

Modern, responsive website for Al Aman Gym, a 24/7 fitness center in Doha, Qatar. Built with React and Tailwind CSS.

## Features
- Hero section with call-to-action buttons
- About the gym with key offerings
- Membership pricing cards
- Gym facilities grid with icons
- Why choose us benefits
- Photo gallery
- Customer testimonials
- Contact section with map, form, and WhatsApp button
- Fully responsive design
- Smooth animations
- SEO optimized

## Deployment on Vercel
1. Push this code to a GitHub repository.
2. Log in to [Vercel](https://vercel.com).
3. Click "Add New Project" and import your repository.
4. Keep default settings (build command: `npm run build`, output directory: `dist`).
5. Click "Deploy".

Your site will be live in seconds!
